A decoupled, event-driven infrastructure that converts real-time voice requirements into production-ready software using multi-agent orchestration.

## 🛠️ Technical Stack
- **Frontend:** React 18, Zustand (Transient Updates), Tailwind CSS, Glassmorphic UI.
- **Backend:** Node.js Gateway, WebSockets, Express.
- **Orchestration:** LangGraph, Python, Redis (Event Bus).
- **Cloud:** Google Cloud Platform (Cloud Run, Artifact Registry).
- **DevOps:** GitHub Actions (CI/CD), pnpm, Turborepo.

## 📜 Governance

- **Architect (Antigravity):** High-level design and validation.
- **Executor (Claude Code):** Low-level implementation and DevOps.
- **Shadow Architect (Sifu):** Observability and documentation.
